## Production Workflow

### Phase 1: Planning (Monday)
1. Review content calendar
2. Select topics from `03_Video_Topics/VIDEO_TOPICS_DATABASE.md`
3. Research topic on appropriate website:
   - Vietnamese: www.apolo.com.vn
   - English: www.apololawyers.com
4. Create script outline

### Phase 2: Script Writing (Monday-Tuesday)
1. Use template from `07_Templates/`
2. Include:
   - Hook (first 3 seconds)
   - Main content (2-5 key points)
   - Call to action
   - Contact info (language appropriate)
3. Review for legal accuracy
4. Save to `04_Scripts/[Language]/`

### Phase 3: Production (Tuesday-Friday)
1. Set up filming environment
2. Record video
3. Capture b-roll if needed
4. Edit video
5. Add captions/subtitles
6. Create thumbnail
7. Export for YouTube (16:9) and TikTok (9:16)

### Phase 4: Publishing (Friday-Saturday)
1. Upload to YouTube
2. Upload to TikTok
3. Fill out metadata:
   - Title (SEO optimized)
   - Description with backlinks
   - Tags/hashtags
   - Thumbnail
4. Schedule or publish
5. Update content calendar
6. Move files to `06_Published/`

### Phase 5: Review (Weekly)
1. Check analytics
2. Respond to comments
3. Note successful topics
4. Plan next week

## AI Content Generation Guidelines

When using AI to write scripts:

1. **Provide context:**
   - Language (Vietnamese or English)
   - Target audience (VN citizens or foreigners)
   - Source website for reference
   - Legal topic from database

2. **Review requirements:**
   - All legal claims must be verified
   - Citations should link to correct website
   - Tone should match brand voice
   - No cross-language source mixing

3. **Template prompt for Vietnamese:**
   ```
   Viết script video về [TOPIC] cho kênh YouTube Công ty Luật Apolo Lawyers.
   - Độ dài: 3-5 phút
   - Trang nguồn: www.apolo.com.vn
   - Đối tượng: Người Việt cần tư vấn pháp luật
   - Bao gồm thông tin liên hệ cuối video
   ```

4. **Template prompt for English:**
   ```
   Write a video script about [TOPIC] for APOLO LAWYERS YouTube channel.
   - Duration: 3-5 minutes
   - Source website: www.apololawyers.com
   - Target audience: Foreigners living in Ho Chi Minh City
   - Include contact information at the end
   ```
