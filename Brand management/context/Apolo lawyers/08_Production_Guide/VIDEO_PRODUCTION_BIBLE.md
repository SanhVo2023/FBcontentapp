### Timeline chuẩn cho video 45-60 giây:

| Phần | Thời gian | Nội dung |
|------|-----------|----------|
| HOOK | 0:00-0:03 | Câu gây sốc/hỏi/số liệu |

| SOLUTION | 0:10-0:40 | 2-3 điểm giải pháp |
| CTA | 0:40-0:45/60 | Liên hệ + Follow |

### Nguyên tắc Shorts:
- **Hook trong 3 giây đầu** - quyết định người xem ở lại hay lướt
- **1 chủ đề = 1 video** - không lan man
- **Nói nhanh, gọn, súc tích**
- **On-screen text LUÔN có** - 80% xem không bật âm thanh
- **CTA ngắn gọn** - chỉ hotline + follow

# PHẦN 10: FILE NAMING CONVENTION

## 10.1 Quy tắc đặt tên file

### Video Files
```
[LANG]_[TOPIC]_[VERSION]_[DATE].mp4

Ví dụ:
VN_LyHonThuanTinh_v1_20260318.mp4
EN_WorkPermit_v2_20260318.mp4
```

### Thumbnail Files
```
THUMB_[LANG]_[TOPIC]_[DATE].jpg

Ví dụ:
THUMB_VN_LyHonThuanTinh_20260318.jpg
THUMB_EN_WorkPermit_20260318.jpg
```

### Script Files
```
SCRIPT_[LANG]_[TOPIC]_[DATE].md

Ví dụ:
SCRIPT_VN_LyHonThuanTinh_20260318.md
```

### B-roll Files
```
BROLL_[CATEGORY]_[DESCRIPTION]_[NUMBER].mp4

Ví dụ:
BROLL_Legal_Courtroom_001.mp4
BROLL_Family_HappyCouple_002.mp4
```
